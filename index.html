About me

I am a developer living in Yarmouth, Nova Scotia. I was born March 1st 2002 in Mississauga Toronto, moving to Nova Scotia in 2015. Around that time I began programming.

In 2020 I began taking game development seriously, focusing on Unity. Then in 2022 I enrolled in college, which introduced me to low level languages, web development & frameworks and best practices.

Today, I am interested in Game and Web Development

For Game Development, I am focused on Unity, I also use Blender to create models, Krita to create textures and FL Studio for audio and music.

For Web Development, I am not focused on one particular technology, and have used Vue, PHP, Django and Node.

However, I am enthusiastic to learn new technologies if required for a role.



Use of AI

Racing Car AI

The goal of this project was to make a racing ai, that will respond to nearby cars and avoid them, follow a racing line and be fully controllable.

The end goal would be to have a full race that the cars will run, which can be watched from trackside cameras like a real race.

Web Development

Django Image Sharing Site

Created September 2025

This project was to make a site similar to the Image Board I created with Vue.

Building off of what I learned from the Microblogging site I made with Django, I also added users, allowing users to delete their content. Finally I added a view count, which goes up every time the post is viewed.

With the users, I gave each user a profile page, with a list of all their posts on the side, and a total view count of all their posts.

Django Microblogging Site

Created August 2025

This project was to make a site with a main board, where a user can create a message and have it displayed, allowing for a wall of text of user generated content.

The Goal of this project was to learn the basics of Django, specifically users, storing and reading data, and routing.

Vue Image Sharing Board

Most Common Names in NS

Created February 2023

As a project for my second semester Web Development class, this project was to use an API to get and display data.

Using a Nova Scotia Open Data API, this website displays the most common names of newborns between 1920 & 2022.
